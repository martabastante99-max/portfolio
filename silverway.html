Silverway Signal Tracker

A self-built system that scores the top 200 global tech companies every day and delivers a ranked report straight to my inbox — before the market opens.

What it does

Every morning, a GitHub Actions workflow triggers a Python script that pulls data on the top 200 tech and software companies globally by market cap. It scores each one across eight parameters and ranks them in a single list delivered by email.

Signals used

Alternative (community momentum):

  • GitHub stars — developers bookmark tools they actually use. 50k+ stars takes years to build.
  • Google Trends (7 days) — rising public interest before it shows up in coverage or price.
  • Hacker News (7 days) — the most senior tech community in the world: engineers, founders, CTOs.
  • Wikipedia pageviews (7 days) — when investors and journalists are researching a company, this spikes.

Valuation fundamentals: P/E ratio, 5-year revenue growth, ROA, Debt/Equity.

Key findings (May 28, 2026)

The highest score was 66%, suggesting the market is generally fairly priced across tech. Standout picks for further investigation: NVDA, ADBE, SMCI, SAP, and KSPI.

What I learned — and what I'd improve

Google Trends returned errors for ~60% of companies, distorting the rankings. I documented this limitation clearly. Planned improvements:

  • Weight scores differently by company type (mature vs. growth)
  • Use multiple APIs per parameter and average them
  • Replace yfinance with a more reliable financial data source
  • Segment results by market cap so large and small caps don't compete on the same scale
